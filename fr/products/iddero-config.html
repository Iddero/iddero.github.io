iddero config

Software de configuración para pantallas táctiles HCx-KNX y para Iddero Home Server

Crea tus proyectos en cuestión de minutos:

  • Importa direcciones de grupo desde ETS
  • Define los planos, zonas y elementos a controlar
  • Parametriza cada elemento o función de forma rápida e intuitiva
  • Asocia objetos de comunicación a direcciones de grupo mediante un cómodo interfaz "arrastrar y soltar" (drag & drop)
  • Transfiere la configuración al dispositivo cómodamente a través de TCP/IP
  • Tanto la visualización como el interfaz web se generan de forma automática, sin necesidad de diseñar cada elemento del interfaz de usuario.
